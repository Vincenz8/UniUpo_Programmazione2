# 1) 

Scrivere un programma che riceve per argomento un numero intero tra 0 e 10 e:
- tramite la funzione buildlist() genera una lista contenente numeri
casuali tra 0 e 50;
- tramite la funzione list_print() visualizza la lista generata
- tramite la funzione list_mean() esegue la media dei nodi;
- scrive sul file lista.txt il contenuto della lista, un nodo per riga.

# 2)
Scrivere un programma che riceve per argomento il nome di un file di testo contenente numeri interi e:

- tramite la funzione buildlist() genera una lista contenente i numeri
all'interno del file;
- tramite la funzione list_print() visualizza la lista generata
- tramite la funzione list_sum_even() esegue la somma dei nodi di
posizione pari (considerare 0 come pari).

# 3)
Scrivere un programma che riceve per argomento una
stringa e:
- tramite la funzione buildlist() genera una lista contenente i singoli
caratteri all'interno della stringa ricevuta come argomento;
- tramite la funzione list_print() visualizza la lista generata
- tramite la funzione list_length() indica il numero di nodi presenti;
- tramite la funzione count_vowels() esegue il conteggio delle vocali
presenti (sia maiuscole che minuscole).
